HyperConvNets¶

Needing to explore different architectures for a given problem is a common use-case. Many a times, it is a trial and error process where one has to try different configurations of a given architecture and then finally select one that suits a particular porblem.

This notebook explores an idea that allows one to explore different architectures in a very efficient way.

A network of multiple convolutional layers for classifying images from CIFAR10 is used as the example.

High-level structure of the contents:

  • Import required packages
  • Create dataloaders for the CIFAR10 dataset
  • Define a HyperConvNet architecture that allows easy architectural design space exploration
  • Train HyperConvNet and explore the validation accuracy for different configurations of increasing sizes
  • Compare the classification performance of the HyperConvNet with a configuration where all the layers are trained

It is a continuation of the exploration that started with HyperNet for classifying hand-written digits from MNIST dataset. A notebook for the same can be found in the repository hosting this notebook HyperNet.

Idea¶

Imagine if convolutional layers of CNNs (convolutional neural networks) could be trained similiarly to HyperNets.

Refer the figure given below. This architecture consists of 5 convolutional layers. Random segments (anchored at the lower end) are trained in each batch. One instance of a training boundary is shown by dashed line. The weights of the kernels at the lower ends are trained more compared to the kernels at top as the training proceeds.

The definition of the model and experimental data is shown in the sections below.

# Function for training a model for one epoch

def train(model, trainloader, criterion, optimizer, extra_model_args = None):

    model.train()

    all_loss = 0
    correct_count, all_count = 0, 0

    for inputs, labels in trainloader:
        # Don't flatten CIFAR10 images into a 3*32*32 i.e. long vector
        #images = images.view(images.shape[0], -1)
    
        # Training pass
        optimizer.zero_grad()
        
        if extra_model_args == None:
            outputs = model(inputs)
        else:
            outputs = model(inputs, extra_model_args)
        
        loss = criterion(outputs, labels)
        
        #This is where the model learns by backpropagating
        loss.backward()
        
        #And optimizes its weights here
        optimizer.step()
        
        # Calculate validation accuracy
        _, predictions = torch.max(outputs.data, 1)
        correct_preds = predictions.eq(labels.data.view_as(predictions))
        # Convert correct_counts to float and then compute the mean
        batch_avg_acc = torch.mean(correct_preds.type(torch.FloatTensor))
        # Compute total accuracy in the whole batch and add to valid_acc
        correct_count += batch_avg_acc.item() * inputs.size(0)
        
        all_loss += loss.item() * inputs.shape[0]
        all_count += inputs.size(0)
    return (all_loss / all_count, correct_count / all_count)

# Function for running a validation run

def validate(model, valloader, criterion, extra_model_args = None):

    model.eval()

    all_loss = 0
    correct_count, all_count = 0, 0

    for inputs, labels in valloader:
        with torch.no_grad():
            if extra_model_args == None:
                outputs = model(inputs)
            else:
                outputs = model(inputs, extra_model_args)
            
            loss = criterion(outputs, labels)
            
            # Calculate validation accuracy
            _, predictions = torch.max(outputs.data, 1)
            correct_preds = predictions.eq(labels.data.view_as(predictions))
            # Convert correct_counts to float and then compute the mean
            batch_avg_acc = torch.mean(correct_preds.type(torch.FloatTensor))

            # Compute total accuracy in the whole batch and add to valid_acc
            correct_count += batch_avg_acc.item() * inputs.size(0)
            
            all_loss += loss.item() * inputs.shape[0]
            all_count += inputs.size(0)
    return (all_loss / all_count, correct_count / all_count)

# Definition of hyper-convolutional layer

class HyperConvLayer(nn.Module):
    def __init__(self, input_shape, out_channel_segment_specs):
        super(HyperConvLayer, self).__init__()

        self._input_shape = torch.Size(input_shape)
        batch_size = input_shape[0]
        num_input_channels = input_shape[1]

        w_in = input_shape[2]
        h_in = input_shape[3]

        # create segments of output channels
        segments = nn.ModuleList()
        w_out = None
        h_out = None

        out_channel_sizes = []
        for out_channel_segment_spec in out_channel_segment_specs:
            num_out_channel_segment = out_channel_segment_spec[0]
            out_channel_sizes.append(num_out_channel_segment)

            k = out_channel_segment_spec[1][0]
            s = out_channel_segment_spec[1][1]
            p = out_channel_segment_spec[1][2]

            w_out_seg = math.floor(1 + (w_in + 2 * p - k)/s)
            h_out_seg = math.floor(1 + (h_in + 2 * p - k)/s)

            if w_out == None:
                w_out = w_out_seg
            else:
                assert w_out_seg == w_out, "All the segments in a layer must have same output width"
            if h_out == None:
                h_out = h_out_seg
            else:
                assert h_out_seg == h_out, "All the segments in a layer must have same output height"

            segments.append(
                nn.Sequential(
                    nn.Conv2d(num_input_channels, num_out_channel_segment, kernel_size=k, stride=s, padding=p),
                    nn.BatchNorm2d(num_out_channel_segment),
                    nn.LeakyReLU(0.02)))

        self._segments = segments

        self._output_shape = torch.Size([batch_size, sum(out_channel_sizes), w_out, h_out])

        for m in self.modules():
            if isinstance(m, nn.Conv2d):
                nn.init.kaiming_normal_(m.weight, mode='fan_out', nonlinearity='relu')
            elif isinstance(m, (nn.BatchNorm2d, nn.GroupNorm)):
                nn.init.constant_(m.weight, 1)
                nn.init.constant_(m.bias, 0)

    def forward(self, x_partial, active_segments_spec = "random"):

        num_partial_channels = x_partial.shape[1]

        x_padding_tail = torch.zeros(x_partial.shape[0], self._input_shape[1] - num_partial_channels, self._input_shape[2], self._input_shape[3])

        x_expanded = torch.cat([x_partial, x_padding_tail], dim=1)

        if active_segments_spec == "random":
            # random selection
            num_active_segments = np.random.randint(1, len(self._segments) + 1)
        elif active_segments_spec == "all":
            num_active_segments = len(self._segments)
        else: # has to be a number
            num_active_segments = active_segments_spec

        # do forward pass only for the requested number of segments
        active_segments = self._segments[0:num_active_segments]

        segment_outs = []
        for active_segment in active_segments:
            segment_outs.append(active_segment(x_expanded))

        return torch.cat(segment_outs, dim=1)
    
    def output_shape(self):
        return self._output_shape

# Definition oc Hyper-Convolutional Neural Network
# It supports instantiation with different configurations e.g.

# model = HyperConvNet([batch_size, 3, 32, 32], [
#     [(16, (3, 2, 1)), (16, (5, 2, 2)), (16, (7, 2, 3))],
#     [(16, (3, 2, 1)), (16, (5, 2, 2)), (16, (7, 2, 3))],
#     [(16, (3, 2, 1)), (16, (5, 2, 2)), (16, (7, 2, 3))],
#     [(16, (3, 2, 1)), (16, (5, 2, 2)), (16, (7, 2, 3))],
#     [(16, (3, 2, 1)), (16, (5, 2, 2)), (16, (7, 2, 3))]],
#     128,
#     10)

class HyperConvNet(nn.Module):
    def __init__(self, input_shape, conv_channels_segment_pecs_list, last_fc_size, num_classes):
        super(HyperConvNet, self).__init__()

        #self._conv_channels_segments_list = conv_channels_segments_list.copy()

        self.conv_layers = nn.ModuleList()

        self.conv_layer_out_shapes = []
        prev_layer_shape = input_shape

        for conv_channel_segments_spec in conv_channels_segment_pecs_list:
            hyper_conv_layer = HyperConvLayer(prev_layer_shape, conv_channel_segments_spec)
            self.conv_layers.append(hyper_conv_layer)
            prev_layer_shape = hyper_conv_layer.output_shape()
            self.conv_layer_out_shapes.append(prev_layer_shape)

        self.flatten = nn.Flatten()

        def prod_shape(shape):
            prod = 1
            for v in shape:
                prod *= v
            return prod

        self.fc_size = prod_shape(self.conv_layer_out_shapes[-1][1:])
        self.last_fc_size = last_fc_size

        self.fc1 = nn.Linear(self.fc_size, last_fc_size)
        self.fc2 = nn.Linear(last_fc_size, num_classes)
        
        self.out = nn.LogSoftmax(dim=1)
        
        for m in self.modules():
            if isinstance(m, nn.Conv2d):
                nn.init.kaiming_normal_(m.weight, mode='fan_out', nonlinearity='relu')
            elif isinstance(m, (nn.BatchNorm2d, nn.GroupNorm)):
                nn.init.constant_(m.weight, 1)
                nn.init.constant_(m.bias, 0)

    def forward(self, x, active_segments_spec = "random"):
        num_layers = len(self.conv_layers)
        for idx in range(num_layers):
            conv_layer = self.conv_layers[idx]

            if active_segments_spec == "random":
                x = conv_layer(x, "random") # will be random
            elif active_segments_spec == "all":
                x = conv_layer(x, "all") # will be random
            else:
                x = conv_layer(x, active_segments_spec[idx]) # whatever is passed

        x = self.flatten(x)
        #print(f"shape after transfer_0 = {x.shape}")

        #assert x.shape[1:] == torch.Size([prod_shape(self.conv_layer_out_shapes[-1][1:])])

        # this x may be partial
        x_padding_tail = torch.zeros(x.shape[0], self.fc_size - x.shape[1])

        x = torch.cat([x, x_padding_tail], dim=1)

        x = self.fc1(x)
        
        x = self.fc2(x)
        
        #print(f"shape after fc_0 = {x_3.shape}")

        y = self.out(x)
        #print(f"shape after out = {y.shape}")
        
        return y

Conclusion¶

It can be observed the single instance of HyperConvNet can approximate the accuracy of baseline modes within 10% of accuracy. It is way off the mark when only first level kernels are used for validation. However, the classification accuracy jumps significantly with activating two levels in each of the layers. This allows one to gauge the trend of baseline models without having to train multiple versions. This seems like an amazing experimental result. As shown in the other HyperNet notebook of only fully connected layers, prunned models should be possible to be extracted.

In short:

  • The HyperConvnet is able to approximate the performance of 5^6 i.e. thousands of concrete instantions of the model
  • The concept of hyperness extends to convolutional layers and it is likely to extend to other kinds of components
  • Demonstrates amazing plasticity and composibilty of components if trained probablistically

Next steps¶

I am so excited to share this work with the ML community. There are some interesting possible continuations of this work. For those, who are learning and want to get deeper in the code, these could serve as nice learning problems.

  • Transfer learning - Use the lower layers trained from one HyperConvNet for a different image dataset
  • Stochastic determination of more potent boundaries (biased towards uisng lessers components) during trainin
  • Applying this concept in the presence of feed-forward layers in architectures such as ResNet and UNet.
  • Train different layers for different fidelities of input and then combine them in modular way
